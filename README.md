# Interior Designer Management System

A management system designed to help interior designers manage quotations, projects, tasks, expenses, materials, and files in a structured and professional way.

This system is built to replace manual workflows such as paper notes and Excel sheets by providing a centralized dashboard for project tracking and financial visibility.


# Features

### Dashboard
- High-level overview of ongoing projects with immediate urgency alerts for critical tasks.
- Quick navigation to key modules.

### Quotations
- Create and manage client quotations.
- Seamlessly convert approved quotations into active active projects.

### Projects
- Track project details and status (Not Started, In Progress, On Hold, Completed).
- Real-time budget vs. actual expense tracking.
- Automatic progress calculation based on task completion.

### Task Management
- Kanban-style task board organized by trade and priority (To Do / In Progress / Done).
- Quick task creation with due dates and trade categorization.

### Expense Tracking
- Log project expenses with standardized categories, dates, and receipt uploads.
- Automatic calculation of total expenses and visibility into budget variance.

### Files & Photos
- Upload, organize, preview, and download project-related files (images, PDFs, documents).

### Project Notes
- Internal module within project detail to add and view site observations and remarks.

### Clients & Materials
- Manage client contact information.
- Master material database for project selection.
